# <a name="unified-api-for-xamarinios-and-xamarinmac"></a>适用于 Xamarin 和 Xamarin 的 Unified API

Xamarin 的 Unified API 允许在 Mac 和 iOS 之间共享代码，并支持具有相同二进制文件的32和64位应用程序。 默认情况下，在新的 Xamarin 和 Xamarin 项目中使用 Unified API。
